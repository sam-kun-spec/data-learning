# Python - Quick Overview

## What is Python?
A high-level, interpreted, general-purpose programming language known for simple and readable syntax.

## Who Made It?
Created by **Guido van Rossum** (Dutch programmer), first released in **February 1991**.
Named after the comedy show **"Monty Python's Flying Circus"**, not the snake.

## Key Features
- Clean, easy-to-read syntax
- Interpreted & dynamically typed
- Cross-platform (Windows, macOS, Linux)
- Massive library ecosystem

## Where is Python Used?
- Web development, Data Science, Machine Learning
- Automation, Cybersecurity, Game development

## Current Version
Python 3.x is actively maintained. Python 2 is dead (EOL 2020).
