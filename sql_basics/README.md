# SQL Practice — Student Performance Analysis

## Dataset
**Students Performance in Exams** — Kaggle  
1000 rows | 8 columns | scores in math, reading, writing

## Questions Answered

| # | Question |
|---|----------|
| Q1 | Average math, reading, writing score by gender |
| Q2 | Top 5 students by math score |
| Q3 | Count of students who completed vs skipped test preparation |
| Q4 | Average math score by parental education level (highest to lowest) |
| Q5 | Students who scored above average in all 3 subjects |

## Tools Used
PostgreSQL, pgAdmin

## Key Findings
- Males score higher in math; females score higher in reading and writing
- Students with master's degree parents have the highest avg math score
- Only 358/1000 students completed test preparation
- 374 students scored above average in all 3 subjects
